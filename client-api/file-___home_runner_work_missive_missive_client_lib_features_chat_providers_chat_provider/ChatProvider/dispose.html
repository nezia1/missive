dispose method

  1. @override
void dispose()

Disposes the provider and closes the WebSocket connection. This method is called when the user logs out, as we need to clean up the resources to avoid sessions getting mixed up.

Usage

provider.dispose();

Implementation

@override
void dispose() {
  _disposed = true;
  _logger.log(Level.FINE, 'Disposing... (this should happen after log out)');
  if (_channel != null) {
    _channel?.sink.close();
  }
  _messagesSubscription?.cancel();
  _userRealm?.close();
  super.dispose();
}
